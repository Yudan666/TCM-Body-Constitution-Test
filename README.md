# 中医体质辨识自测（TCM Body Constitution Test）

这是一个基于 **中医九种体质理论** 的前端自测问卷页面，采用纯 HTML / CSS / JavaScript 实现。  
所有计算均在本地浏览器完成，不依赖后端服务，也不会收集或上传任何用户数据。

---

## ✨ 功能简介

- 📋 基于 60+ 标准化题目进行体质辨识  
- 🧠 前端即时计算体质得分与判定结果  
- 📊 可视化展示各体质倾向（条形图）  
- 🍵 提供基础调理建议与生活方式提示  
- 🔒 完全本地运行，隐私安全，不保存任何填写数据  
- 📱 适配移动端，支持 iOS / Android 浏览器访问  

---

## 🚀 使用方式

1. 打开页面（GitHub Pages 链接）
2. 选择性别（可选）
3. 完成问卷填写
4. 即时查看体质分析结果与建议

无需注册、无需登录、无需网络提交。

---

## 🛠 技术说明

- 前端：HTML + CSS + 原生 JavaScript  
- 架构：单文件单页应用（SPA 风格）  
- 数据处理：全部在客户端完成  
- 后端：无  
- 适用场景：个人测试、学习演示、小范围分享
